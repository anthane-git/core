# Headline

Headline is used to render semantic HTML heading elements.

<a
	href={
		'https://anthane-git.github.io/core-elements/?path=/story/text-typography--component-headline'
	}
	target={'_blank'}>
	Storybook Link

<br />

## Import

To implement headline component into your project you'll need to add the import:

```tsx
import { Typography } from '@anthane/core-elements';
```

## Usage

After adding import into your project you can use it simply like:

```tsx
<Typography.Headline size="Medium">
	The quick brown fox jumps over the lazy dog
</Typography.Headline>
```
