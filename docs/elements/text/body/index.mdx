# Body

Body is used to render semantic HTML paragraph elements.

## Import

To implement Body component into your project you'll need to add the import:

```tsx
import { Typography } from '@anthane/core-elements';
```

## Usage

After adding import into your project you can use it simply like:

```tsx
<Typography.Body>The quick brown fox jumps over the lazy dog</Typography.Body>
```
